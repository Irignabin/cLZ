## ğŸš€ Getting Started

### Prerequisites

- PHP 8.2+, Composer (for backend)
- Node.js, npm/yarn (for frontend)
- MySQL/MariaDB or equivalent database
- Git

---

### Setup Backend

See detailed backend setup instructions:

[docs/setup.md](./blood-donation-backend/docs/setup.md)

Quick summary:

```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
# configure .env database and app URL
php artisan migrate --seed
php artisan serve
```

---

### Setup Frontend

Navigate to frontend folder and install dependencies:

```bash
cd frontend
npm install
npm run start
# or
yarn dev
```

Adjust API base URLs in frontend environment config as needed.
