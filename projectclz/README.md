# Smart Blood Bank - Life Line

A modern web application that connects blood donors with those in need. Built with React, Material-UI, and Leaflet maps.

## Features

- **User Authentication**: Secure login and signup system
- **Donor Registration**: Easy process to become a blood donor
- **Interactive Map**: Find donors near you with real-time location data
- **Blood Type Filtering**: Search donors by specific blood types
- **Emergency Notifications**: Broadcast urgent blood requests
- **Responsive Design**: Works seamlessly on all devices

## Tech Stack

- **Frontend**:
  - React 18
  - TypeScript
  - Material-UI v5
  - React Router v6
  - Leaflet Maps
  - Emotion (for styled components)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd blood-donation-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
